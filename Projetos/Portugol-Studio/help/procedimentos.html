Procedimentos



Procedimentos são subrotinas que tem por objetivo executar alguma operação especifica, não retornando valor algum.

Exemplo 1

Programa em que o usuario digita um numero qualquer no programa principal e a frase se repete N vezes no Procedimento 



programa
{
	funcao inicio(cadeia parametros[])
	{
	  inteiro vezes 

	  escreva ("Digite um numero ")
	  leia (vezes)

	 frase(vezes)

	  escreva ("\nDevolta ao programa principal\n\n")

	}
	
     funcao frase (inteiro vezes) {
     	escreva ("\nA Frase vai repitir ",vezes," vezes\n\n")

     	para (inteiro i = 0; i < vezes; i++) {
     		escreva ("Oi!\n")
     	}
     }
}
